## [1.3.1] - 2024-09-17
### 新增内容
- “添加多项日志” 功能添加了 “修饰消息头” 的额外功能选项
- “版本选项” 右键菜单添加 “复制日志” 选项，方便到时候一键 Git 提交
- 添加状态栏的对 Git 提交 状态的支持
